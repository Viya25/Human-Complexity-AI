# Human Complexity AI

## Overview

Human Complexity AI is an Explainable Machine Learning framework designed to identify students at academic risk using behavioral, academic, and lifestyle indicators.

## Features

- Academic Risk Prediction
- Random Forest Classifier
- Explainable AI using SHAP
- Personalized Recommendations
- Interactive Streamlit Dashboard

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- SHAP
- Streamlit
- Matplotlib

## Results

- Logistic Regression Accuracy: 63.29%
- Random Forest Accuracy: 64.56%

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app/streamlit_app.py
```
